Try something like:

var matches = text.match(/\[\[.*?\]\]/);

matches[0] will be the matched string.

Since the DOTALL PCRE option isn't supported in Javascript, you'd have to use a regular expression like that:

var matches = text.match(/\[\[(?:\s|.)*?\]\]/);
